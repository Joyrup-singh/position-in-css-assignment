Absolute Relative
The element is positioned absolutely to its first positioned parent. The element is positioned relative to its normal position.
Syntax:position: absolute; Syntax:position: relative;
The element is positioned nearest to the first relatively or absolutely positioned parent element. When an attribute left:20px is added the element moves 20px to the right from its normal position. The element effects no other element on the layout. Other content will not be adjusted to fit into any gap left by the element.
When there is no parent element the absolutely positioned element is positioned directly to the HTML page itself The child of this element can be positioned within this block only. So position:relative has its limitations.
